# Mini Catálogo de Celulares - Novo Gama-GO 📱🗺️

Este projeto é uma aplicação web simples e responsiva que funciona como um guia local de lojas e assistência técnica de celulares na região de Novo Gama, Goiás. Ele permite que usuários encontrem rapidamente estabelecimentos próximos com base no nome do local.

> **Status do Projeto:** Concluído ✅

---

## 🚀 Funcionalidades

* **Catálogo Visual:** Exibição de estabelecimentos em formato de *cards* modernos e organizados em *grid*.
* **Busca em Tempo Real:** Sistema de filtragem dinâmico em JavaScript que atualiza o catálogo instantaneamente enquanto o usuário digita o nome da loja.
* **Design Responsivo:** Layout adaptável para proporcionar uma boa experiência de uso tanto em dispositivos móveis (smartphones) quanto em desktops.
* **Alertas de Informação:** Botões "Ver mais" que exibem os dados de endereço do local através de alertas nativos do navegador.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web, sem a necessidade de *frameworks* externos:

1. **HTML5:** Estruturação semântica e acessível do conteúdo.
2. **CSS3:** Estilização moderna, uso de Flexbox e CSS Grid para layout responsivo.
3. **JavaScript (Vanilla):** Lógica de manipulação do DOM para o sistema de busca e filtragem.

## ♿ Acessibilidade e Semântica

Um dos pontos fortes deste projeto é o cuidado com a semântica e acessibilidade:

* Uso de tags semânticas HTML5 (`<header>`, `<main>`, `<section>`, `<article>`).
* Uso de atributos ARIA (`aria-labelledby`, `aria-live`, `aria-describedby`) para melhorar a experiência com leitores de tela.
* Navegação por teclado garantida através de `tabindex` nos *cards* e estados `:focus` visíveis no CSS.
